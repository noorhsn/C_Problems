# C_Problems

A repository to sync and store LeetCode practice code and challenges.

## Repository Structure

```
C_Problems/
├── Easy/           # Easy difficulty problems
├── Medium/         # Medium difficulty problems
├── Hard/           # Hard difficulty problems
└── README.md
```

## How to Use

1. Solutions are organized by difficulty level
2. Each solution file is named with the problem number and title (e.g., `0001_Two_Sum.c`)
3. Each solution includes the problem description as comments at the top

## Contributing

Feel free to add your own solutions or improve existing ones!
